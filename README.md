# small-library-app
##Overview
A simple library app that lets you keep track of your books
###Features
- You can add a book
- You can delete a book
- Modal to add new books
- Change read status of book
- 
###Tools
- Visual Studio Code
- Linux Terminal
- Git and Github
- ###Learning outcomes
- How to use CSS grid to create a card gallery
- How to toggle classes
